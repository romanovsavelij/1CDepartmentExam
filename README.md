# 1CDepartmentExam

Отбор на кафедру 1С в МФТИ.

## Установка

```shell script
pip3 install -r requirements.txt
```

## Запуск

```shell script
python3 main.py
```

## Тесты

```shell script
python3 -m unittest tests/*
```
